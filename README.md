# Network Performance Lab

Professional HTTP throughput analyzer and load testing suite.

## Features
- Real-time request latency measurement
- Configurable concurrency levels
- HTTP method support (GET, POST, HEAD)
- Live statistics dashboard

## Usage
1. Open `index.html` in browser
2. Configure target endpoint
3. Set concurrency and duration
4. Run load test

## Legal
Use only for authorized testing on systems you own or have permission to test.
